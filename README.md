# InfAbi26 – Lernplan Informatik-Leistungskurs Abitur 2026

Interaktiver HTML-Lernplan für den Informatik-Leistungskurs (Abiturjahrgang 2026).

## Öffnen

Die Datei `index.html` direkt im Browser öffnen (kein Server erforderlich).

## Inhalt

Eine strukturierte Übersicht der Abitur-relevanten Themen des Informatik-Leistungskurses,
dargestellt als interaktive Webseite.

## Technologie

- Reines HTML (kein JavaScript-Framework, kein Server)
- Kann lokal oder per GitHub Pages bereitgestellt werden

## Für Schüler

Dieser Lernplan gibt einen Überblick über alle prüfungsrelevanten Themen.
Er ist das zentrale Navigationsinstrument für die Abiturvorbereitungsphase.
